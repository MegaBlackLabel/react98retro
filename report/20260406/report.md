# react98retro 実装レポート（2026-04-06）

## プロジェクト概要

Windows 98 / Windows 95 スタイルの UI を React で再現するコンポーネントライブラリ。  
`98.css` をベースに、ドラッグ・リサイズ可能なウィンドウと本格的なファイルエクスプローラーを実装。

---

## 実装済み機能

### 1. UI コンポーネント（`src/components/`）

| コンポーネント | 概要 |
|---|---|
| `Window` | ドラッグ・リサイズ・最小化・最大化対応のウィンドウ本体 |
| `TitleBar` | タイトルバー（アイコン・タイトル・ウィンドウ操作ボタン） |
| `StatusBar` / `StatusBarField` | ウィンドウ下部ステータスバー |
| `Button` | Win98 スタイルのボタン |
| `Checkbox` | チェックボックス |
| `OptionButton` | ラジオボタン |
| `Dropdown` | ドロップダウンセレクト |
| `TextBox` | テキスト入力欄 |
| `TextArea` | 複数行テキスト入力 |
| `Slider` | スライダー |
| `GroupBox` | グループボックス（フィールドセット風） |
| `FieldRow` | ラベル＋入力のペアレイアウト |
| `MenuBar` | メニューバー（サブメニュー・セパレーター・チェック対応） |
| `Menu` | フローティングメニュー |
| `Toolbar` | ツールバー（ボタン・セパレーター・SplitButton対応） |
| `SplitButton` | ドロップダウン付きボタン |
| `AddressBar` | アドレスバー（ナビゲーション入力） |
| `TreeView` | ツリービュー（折り畳み可能） |
| `Tabs` | タブパネル |
| `TableView` | 一覧テーブル（列ヘッダー付き） |
| `Win98Provider` | `98.css` をスコープ付きで注入するプロバイダー |

### 2. カスタムフック（`src/hooks/`）

| フック | 概要 |
|---|---|
| `useDraggable` | Pointer Events ベースのドラッグ実装。外部 position state との共有に対応 |
| `useResizable` | 8方向リサイズ対応。ドラッグと position state を共有し、左・上端リサイズでも位置がずれない |
| `useWindowManager` | 複数ウィンドウの最小化・最大化・フォーカス（z-index）・クローズを管理 |

### 3. アイコンシステム（`src/icons/`）

- Chicago95 アイコンセット（GPL-3.0）を **jsDelivr CDN 経由**で参照することでバンドルサイズへの影響ゼロ
- ドライブ・フォルダ・ファイル・ツールバーアクション・シェル用など **25種以上** のアイコンを定義
- `getFileIcon(filename)` で拡張子からアイコンを自動解決

### 4. ファイルエクスプローラー Feature（`src/features/file-explorer/`）

| ファイル | 概要 |
|---|---|
| `useFileSystem` | ファイルシステム状態管理。前後履歴・上へ移動・選択・パス表示に対応 |
| `FileTree` | 左ペインのツリービュー（フォルダ展開・ナビゲーション） |
| `FileList` | 右ペインのファイル一覧（詳細ビュー・複数選択） |
| `FileExplorer` | 上記すべてを組み合わせた完全なウィンドウコンポーネント |

**FileExplorer の実装済み要素：**
- タイトルバー付き Window 上で動作
- メニューバー（ファイル・編集・表示・お気に入り・ツール・ヘルプ）
- ツールバー（戻る・進む・上へ・切り取り・コピー・貼り付け・削除・プロパティ・表示）
- アドレスバー
- ツリー＋リストの 2 ペインレイアウト（ドラッグでスプリッター調整可能）
- ステータスバー（オブジェクト数・選択ファイルサイズ・空き領域）
- デフォルトファイルシステム（マイコンピュータ → C: → WINDOWS, Program Files など）

### 5. ライブラリ設定

- **ビルド**: Vite + `vite-plugin-dts` でライブラリビルド（ESM + UMD）
- **テスト**: Vitest + Testing Library + Playwright（カバレッジ対応）
- **Storybook**: 全主要コンポーネントにストーリー完備
- **パッケージ公開先**: GitHub Packages (`@megablacklabel/react98retro`)
